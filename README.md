## Controls
| Key  | Action                                        |
| ---- | --------------------------------------------- |
| ESC  | Quit the program or stop the current sorting  |
| R    | Shuffle the list                              |
| B    | Bubblesort                                    |
| Q    | Quicksort                                     |
